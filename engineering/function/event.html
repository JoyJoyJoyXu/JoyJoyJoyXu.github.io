# 合成事件

event 是 React 封装的,可以看 proto.constructor 是 SyntheticEvent,模拟出来 DOM 事件所有能力

event.nativeEvent 是原生事件对象 所有的事件,都被挂载到 document 上, 然后派发事件 dispactchEvent

# 为什么要合成机制

更好的兼容性和跨平台

挂到 document，减少内存消耗,避免频繁解绑

方便事件的统一管理(如事物机制)
